## ⚡ What is SafayaMetaFix?
SafayaMetaFix is a highly secure, next-generation web application designed to wipe EXIF data, GPS location tags, and device signatures from your video files. Whether you are a privacy advocate, a journalist, or just someone who values digital anonymity, this tool ensures your media is clean before you share it online.


> `![SafayaMetaFix Home](./screenshot-home.png)`

---

## 🧠 How It Works Under the Hood
Unlike traditional converters that force you to upload your personal files to a remote server, **SafayaMetaFix processes everything inside your browser.** Powered by **WebAssembly (WASM)** and a localized **FFmpeg Core Engine**, your files are loaded into a secure virtual container in your RAM. The engine scrambles internal file bit-depths to reset hash values and wipes all metadata tracking points. **No data ever leaves your device.**

## 🚀 Key Features & Benefits
* **🕵️‍♂️ Total Anonymity (Zero-Upload):** 100% client-side processing. We don't have servers to store your videos even if we wanted to.
* **🎥 Drag & Drop Payload System:** A smooth, interactive UI with real-time uploading and cleaning progress indicators.
* **🧬 DNA Scramble:** Modifies the file hash to create a mathematically distinct "Ghost File."
* **📬 Secure Comm Channel:** Built-in encrypted contact portal integrated via Formspree for direct communication with the core team.
* **⚡ Lightning Fast:** Utilizes your device's own CPU/GPU power via WebAssembly.

## 🛠️ Operational Protocol (How to Use)
Using SafayaMetaFix is as simple as 1-2-3:
1. **Inject Media:** Go to the Dashboard and Drag & Drop your target video (MP4, MOV, WebM).
2. **Scrub Engine:** Wait a few moments as the FFmpeg core executes the metadata mutation script. Watch the real-time progress bar.
3. **Export Ghost:** Once processing reaches 100%, click "Download Ghost File" to retrieve your untraceable video.

## 💻 Developer Setup (How to Clone)
Want to run SafayaMetaFix locally or contribute to the core? Follow these steps:

**1. Clone the repository:**
```bash
git clone [https://github.com/vinodkumarjaipal/safaya-meta-fix.git](https://github.com/vinodkumarjaipal/safaya-meta-fix.git)
```

**2. Navigate into the directory:**
```bash
cd safaya-meta-fix
```

**3. Install dependencies:**
```bash
npm install
```

**4. Start the development server:**
```bash
npm run dev
```
*The core engine will now be running at `http://localhost:3000`*
